# Instance types

Use Multiplayer Play Mode to run multiplayer scenarios from within the Unity Editor using different instance types. When you set up a Play Mode Scenario, you can create the following types of instances, each serving different testing and development purposes.

| **Topic**                                                       | **Description**                                                                                                                                                                                           |
|-----------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **[Editor instances](main-and-additional-editor-instances.md)** | The Editor is your primary workspace and acts as the first player or server. Additional instances are clones of the Editor that run in separate processes. |
| **[Local instance](local-instance.md)**                         | Separate, standalone builds for your game that can be a client or server.                                                                                                                                 |
