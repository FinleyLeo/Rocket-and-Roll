## Scenario examples

### Common scenarios

Typical multiplayer configurations include:

- **Main Editor host + additional Editor client (direct IP)**: Main Editor acts as host with additional Editor as client using direct IP connection.
- **Main Editor host + additional Editor client (Relay)**: Main Editor acts as host with additional Editor as client using [Unity Relay](https://docs.unity.com/ugs/manual/relay/manual/introduction).
- **Main Editor host + local client (Relay)**: Main Editor acts as host with local build client using [Unity Relay](https://docs.unity.com/ugs/manual/relay/manual/introduction).
- **Main Editor host + local Android client (Relay)**: Main Editor acts as host with Android device client using [Unity Relay](https://docs.unity.com/ugs/manual/relay/manual/introduction).
- **Main Editor client + local server (direct IP)**: Main Editor as client connecting to local server build via direct IP.

### Advanced scenarios

More complex multiplayer configurations include:

- **Main editor client + additional Editor client (Slow Network Sim)**: Main Editor acts as client alongside additional Editor client with network simulation to test slow network conditions.
